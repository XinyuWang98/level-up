## 🚀 核心工作流：八大阶段快速上手 (Quick Start)

本系统适用于任何支持 Agent 功能的现代 IDE（如 Cursor、Windsurf、VS Code + Copilot 等）。克隆项目后，**请按以下顺序唤醒你的 AI 导师并使用内置 Workflow**：

### 阶段 1: 扫描盲区 (Benchmark)
- **寻找目标**: 找到你心仪岗位的 JD (Job Description)。
- **触发指令**: 在对话框输入 `/generate_gap_analysis` 并提供 JD。Agent 将生成客观的技能差距矩阵，保存在 `03_职业规划` 目录。

### 阶段 2, 3, 4: 实战进阶与知识沉淀 (Hands-on Labs & Knowledge Base)
- **定制练习集**: 输入 `/generate_notebook`，Agent 会主动向你要 Kaggle API，并检索真实的商业数据集出题（存放于 `00_教学课件` 或 `01_专项专栏`）。
- **重识视频流**: 遇到认知盲区时，触发 `/recommend_resources`，让 Agent 联网搜索高赞解说视频（从根源拒绝模型出现幻觉伪造链接）。
- **织网与归档**: 练习完成后，输入 `/update_cheatsheet` 将今日所学结构化归档到 `02_速查手册` 当中。
- **每日双重复盘**: 临睡前输入 `/daily_progress` 和 `/daily_review`，生成客观视角的学习日志（存放于 `06_复盘与日志示例`）。

### 阶段 5 & 6: 求职冲刺与面试模拟 (Interview Prep & Resume Builder)
- **真实笔试卷**: 输入 `/generate_exam`，Agent 会先通过 Web Search 检索目标公司的历年真题，再为你量身定制复杂实战套题（存放于 `05_笔试模拟`）。
- **高压面试场**: 输入 `/mock_interview` 给出目标公司和简历。Agent 会输出一套专为**具有语音交互能力的第三方大模型 (如豆包专家模式、ChatGPT Voice 等)** 准备的严苛判卷 Prompt。请拿着它去和语音模型真刀真枪地对练，练完后再把薄弱环节反馈发回给这里的 Agent 帮你制定复习计划（复盘放在 `04_面试练习`）。
- **千人千面简历**: 确认技能达标后，通过 `/generate_resume` 基于不同岗位的 JD 快速组装投递专用的简历。

### 阶段 7: 投递追踪 (App Tracker)
- **管理面试漏斗**: 使用 `/update_applications` 更新和记录面试表、考察点与反馈，打造可视化的求职看板。

### 阶段 8: 无尽复用 (Knowledge Reuse)
- **职场军火库**: 面试成功后，这套系统不会像普通网课一样被埋没。你沉淀在 `02_速查手册` 里的 40+ 篇文档将成为你入职后最顺手的知识库底座。你可以继续在新的业务挑战中用 AI 帮你探索未知并不断壮大它。

## 🙋 FAQ

**Q: 这个系统适合什么水平的人？**  
A: 非常适合有一定编程/SQL基础，想系统性冲刺大厂数据分析高级岗位 / 数据科学岗位的求职者。如果你是完全零基础，建议先借助 AI 学完 Python 基础语法。

**Q: 必须使用特定的 IDE 吗？**  
A: 完全不需要。不管是 Cursor、Windsurf 还是 VS Code (搭配 Copilot/Continue)，都可以无缝接入。系统真正的精髓在于 `.agent/` 下的提示词工程与你在本地维护的 Markdown 习惯。

**Q: 我不是做算法或数分的，这套系统对我有用吗？**  
A: 绝对有。这里的底层方法论是跨学科的：**Benchmark（定位盲区） → 学习沉淀 → 实战模拟 → 求职追踪**。你只需要稍微修改 `.agent/rules/` 里的角色设定，把它变成你的“产品经理”、“Java开发”或“独立开发者”专属导师即可。
