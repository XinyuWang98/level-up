# 🤖 资深数分 + AI 开发者：基于 Google Antigravity 的求职操作系统 (v2.0)

> **项目愿景**：我不只是在找工作，我是在写一套“求职算法”。以 **Senior Data Analyst + AI Developer** 的视角，基于 **Google Antigravity** 打造一套集知识沉淀、动态叙事、自动化投递于一体的闭环操作系统。

---

## 🛠️ 1. 模块化系统架构 (Modular Architecture)

遵循“单一事实来源 (SSOT)”与“高内聚低耦合”原则，将系统拆解为三大核心模块：

### 🧬 A 模块：原子化武器库 (Atomic Tech Vault)
*   **物理位置**：[`02_速查手册/docs/`](file:///Users/catherinewang/Documents/GitHub/level-up/02_速查手册/docs/)
*   **功能**：沉淀统计学、因果推断、A/B 实验、机器学习的**底层原理 + 工业级代码实现**（如 [15_causal_inference.md](file:///Users/catherinewang/Documents/GitHub/level-up/02_速查手册/docs/15_causal_inference.md)）。
*   **价值**：将复杂模型（如 PSM-DID, CUPED）原子化，实现不同场景下的秒级检索与知识调用。

### 🧩 B 模块：上下文对战卡片 (Contextual Combat Cards)
*   **物理位置**：[`02_速查手册/docs/17_interview_prep.md`](file:///Users/catherinewang/Documents/GitHub/level-up/02_速查手册/docs/17_interview_prep.md)
*   **功能**：对 JD 进行结构化解析，建立“业务问题 -> 历史项目 -> 技术方案”的映射矩阵，涵盖了 [A/B 实验专项](file:///Users/catherinewang/Documents/GitHub/level-up/02_速查手册/docs/17a_interview_ab.md) 等。
*   **价值**：确保护城河技术点精准打击公司痛点。

### 📊 C 模块：全链路数据追踪 (Full-Funnel Tracker)
*   **物理位置**：[`02_速查手册/docs/Resume_Tracker.md`](file:///Users/catherinewang/Documents/GitHub/level-up/02_速查手册/docs/Resume_Tracker.md)
*   **功能**：作为系统的 **SSOT (Single Source of Truth)**，记录从“冷启动”到“谈薪成功”的所有环节数据。
*   **价值**：基于漏斗转化数据进行策略对齐，例如利用 [`Narrative_Mapping.md`](file:///Users/catherinewang/Documents/GitHub/level-up/03_职业规划/Narrative_Mapping.md) 进行叙事架构的动态微调。
