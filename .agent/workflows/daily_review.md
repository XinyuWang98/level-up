# 每日复盘工作流 (Dual-Perspective Review)

## 1. 核心原则 (Rule 0 & No PUA)
- **双向评价**：区分 Agent Self-Reflection (教学质量) 和 Candidate Performance (求职准备度)。
- **Metric-Driven**：使用 S/A/B/C 评级，必须给出量身定制的改进点。

## 2. 报告结构
- **🟢 Part 1: 求职准备度 (Candidate Performance)**：逻辑严谨性、业务直觉、高阶方法论（因果/AB）应用。
- **🔴 Part 2: 教学/Agent 质量 (Agent Audit)**：案例是否脱离现实、漏讲关键点、主动性评估。
- **🔵 Part 3: 针对 P7+ 的改进计划**：针对 JD 痛点的专项强化。

## 3. 注意事项
- **隐私**：不包含个人真实信息。
- **文件名**：`XX_Daily_Review_YYYYMMDD.md` 存放在 `06_复盘与日志示例/` (或相应目录)。
