# 速查手册审查工作流

## 审查重点

### A. 展示规范 (Rule 0)
1. **中英文空行**：段落与列表间必须有空行。
2. **符号统一**：代码外使用 `→` 而非 `->`。
3. **Latex 渲染**：校验 `$..$` 和 `$$..$$`。

### B. Mermaid 图规范 (双主题兼容)
- **描边**：必须带 `stroke:#深色,stroke-width:2px`。
- **文字**：统一白色 `color:#fff`。
- **色相**：中等亮度色（如蓝色 `#0969da`）。

### C. 业务对标
- 每个 Cheatsheet 必须有“业务汇报话术模板”及“面试 Q&A”。
- 使用 `??? question` 折叠格式。

## 注意事项
- 文件路径统一从 `02_速查手册/docs/` 开始。
- **隐私**：示例中不得出现特定公司名称。
