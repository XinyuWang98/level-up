# ✍️ 笔试模拟

> Agent 根据目标岗位级别生成的笔试题集

## 📖 这个模块做什么？

这里存放 AI Agent 生成的笔试模拟题，覆盖高级数据分析师岗位的核心技能。每个 Notebook 聚焦一个专题，从选择题到编程实战。

## 📂 文件说明

| 文件                                       | 类型   | 内容                |
| :----------------------------------------- | :----- | :------------------ |
| `高级DA笔试模拟_00_选择题速刷30题.md`      | 选择题 | 30 道快速概念检验   |
| `高级DA笔试模拟_01_数据清洗与处理.ipynb`   | 编程   | Pandas 数据清洗实战 |
| `高级DA笔试模拟_02_AB实验与因果推断.ipynb` | 编程   | A/B 测试 + 因果推断 |
| `高级DA笔试模拟_02_Pandas底层机制.ipynb`   | 编程   | Pandas 高阶避坑     |
| `高级DA笔试模拟_03_AB实验进阶.ipynb`       | 编程   | CUPED/Delta Method  |
| `高级DA笔试模拟_04_SQL编程实战.ipynb`      | 编程   | 复杂 SQL 查询       |
| `高级DA笔试模拟_05_统计学与假设检验.ipynb` | 编程   | 统计推断            |
| `高级DA_PySpark_Pandas_上机练习.ipynb`     | 编程   | PySpark 大数据处理  |

## 🚀 如何使用？

### 生成新的笔试题

```
请根据 [目标岗位级别] 生成一套笔试模拟题，
覆盖以下技能：Pandas、SQL、统计学、A/B 测试。
难度要求：中高级，包含边界情况和陷阱题。
```

### 推荐刷题顺序

```
选择题速刷(热身) → 数据清洗 → Pandas底层 → SQL → 统计学 → AB实验 → 因果推断
```

## 💡 来自作者的经验

- **限时做题**：给自己设 45 分钟倒计时，模拟真实笔试压力
- **先做选择题热身**：快速回顾概念，进入状态
- **SQL 题用纸笔先写**：面试时不一定有 IDE，练习手写 SQL 的能力
