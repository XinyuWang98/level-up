# 📓 复盘与日志示例

> 记录学习历程、总结思维模型与双重视角的高效复盘模板

## 📖 这个模块做什么？

该目录下集中存放了学习阶段的高阶复盘总结。在学习复杂的（如 Pandas / AB 测试 / 因果推断等）知识点时，借助 Agent 从学生和导师的双重视角 (Student & Mentor) 客观审视掌握情况。这些文件剥离了特定的开发时间限制，可被作为日常学习习惯的养成模板。

## 📂 文件说明

| 文件                             | 说明                                 |
| :------------------------------- | :----------------------------------- |
| `Day1_学习复盘报告_Example.md`   | 完整学习闭环的复盘总结（案例）       |
| `Day2_Pandas思维复盘_Example.md` | 针对特定技术栈（Pandas）的思维沉淀   |
| `Day2/Day3_复盘报告_Example*.md` | 日常学习与技术难题攻克复盘（脱敏后） |
| `Daily_Review_Example_*.md`      | 包含 Agent 导师视角的每日综合教与学  |

## 🚀 如何使用？

在每天完成一定量的知识点打卡或通过 `Hands-on Labs` 实战代码练习后，可以向 Agent 发送以下指令触发复盘机制：
（配合本项目核心工作流 `/daily_progress`，要求其审视每日学习成果）

```
请执行 /daily_progress 工作流，总结我今天完成的 Notebook，盘点我还存在的知识 Gap，并为我输出一份格式对齐这里的每日复盘报告。
```

## 💡 来自作者的经验

- **定期整理思路**：双重视角复盘是非常好的习惯，能有效突破初学者的信息茧房盲区。
- **关联 Gap**：养成把复盘中的不足立刻反映到你的职业规划 Gap 矩阵文件里的习惯（配合 `/update_gap`）。
