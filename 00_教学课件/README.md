# 📚 教学课件

> Agent 生成的分级练习 Notebook + 每日学习复盘报告

## 📖 这个模块做什么？

这里存放的是 AI Agent 根据你的学习进度和 Gap Analysis 自动生成的教学材料。包括两类内容：

1. **Jupyter Notebook 练习** — 系统化的分级练习，从基础语法到复杂业务逻辑
2. **每日复盘报告** — Agent 对你每天学习表现的客观评估

## 🚀 如何使用？

### 生成新 Notebook

使用 `/generate_notebook` workflow，Agent 会：

1. 检查你的当前学习阶段（通过 Gap Analysis）
2. 选择合适的真实数据集（优先 Kaggle，禁止 mock 数据）
3. 生成分级挑战：
   - **Level 1**: 基础语法（"Hello World" 级别）
   - **Level 2**: 业务逻辑（真实世界场景）
   - **Level 3**: 边界情况（处理空值、重复项、复杂逻辑）
4. 附带参考答案（折叠隐藏）

### 生成复盘报告

使用 `/daily_review` workflow，Agent 会从客观视角评估你当天的学习表现，指出薄弱点。

## 💡 来自作者的经验

- **不要跳过 Level 1**：很多看似简单的语法，在面试压力下容易犯错
- **复盘报告的价值在于"打脸"**：Agent 会指出你以为掌握了但实际没有的知识点
- **建议每天 2-3 个 Notebook**，不要贪多，消化比刷量重要
