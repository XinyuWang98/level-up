# 🎯 职业规划

> Benchmark 能力体检 → Gap Analysis → 简历定制 → 投递追踪

## 📖 这个模块做什么？

这是学习系统的"指南针"——先明确目标岗位要什么，再规划怎么学。包含四大核心功能：

1. **Gap Analysis（能力体检）** — 对照目标 JD，列出你的技能差距
2. **简历定制** — 根据不同 JD 让 Agent 定制化优化简历
3. **JD 分析** — 拆解 JD 要求，识别核心技能和加分项
4. **投递追踪** — 记录投递进度和面试结果

## 🚀 如何使用？

### Step 1: 创建你的 Gap Analysis

```
请根据以下 JD，分析我的当前技能水平和目标岗位的差距，
按以下维度生成 Gap Analysis 矩阵：
- 技能名称
- 目标水平（JD 要求）
- 当前水平（我的自评）
- Gap 等级（🔴 大 / 🟡 中 / 🟢 小）
- 学习优先级

[粘贴你的目标 JD]
```

### Step 2: 每周更新 Gap

使用 `/update_gap` workflow，让 Agent 根据你这周学的内容重新评估差距：

```
/update_gap
本周我完成了 AB 测试的 CUPED 方差缩减练习，请更新 Gap Analysis。
```

### Step 3: 投简历前定制简历

使用 `/generate_resume` workflow：

```
/generate_resume
请根据以下 JD 定制我的简历，突出匹配的技能和项目经验。
[粘贴目标 JD]
```

**策略**: 维护一份通用版简历 + 针对每个目标岗位生成定制版。

## 💡 来自作者的经验

- **Gap Analysis 是最重要的文件**：它决定了你每天该学什么，避免"感觉都在学但没有方向"
- **每周让 Agent 更新 Gap**：看到🔴变🟡变🟢的过程非常有成就感
- **简历定制不要改太多**：核心经历不变，只调整措辞重心和顺序
