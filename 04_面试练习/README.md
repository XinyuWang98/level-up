# 🎤 面试练习

> 让 AI Agent 扮演面试官，进行多轮技术面试模拟

## 📖 这个模块做什么？

这里存放 AI 模拟面试的总结报告。Agent 会扮演面试官，根据你的简历和目标岗位进行多轮追问，然后生成总结报告帮你查漏补缺。

## 📂 文件说明

| 文件                              | 说明                             |
| :-------------------------------- | :------------------------------- |
| `模拟面试总结报告_示例.md`        | 完整的模拟面试总结报告（脱敏后） |
| `AB实验面试练习_Word排版示例.txt` | A/B 实验面试问答练习记录         |

## 🚀 如何使用？

### 发起模拟面试

```
请扮演 [目标公司] 的高级数据分析师面试官，
根据我的简历对我进行技术面试。
重点考察以下方向：
1. 项目深挖（STAR 法则追问）
2. A/B 测试方法论
3. 因果推断实际应用
4. SQL/Pandas 编程

请一次问一个问题，根据我的回答决定是否追问。
```

### 面试后生成总结

```
请生成本次模拟面试的总结报告，包括：
1. 每道题的评分（1-5）
2. 我的回答有哪些亮点和不足
3. 建议的改进话术
4. 需要补充学习的知识点
```

## 💡 来自作者的经验

- **多用不同 AI 模拟**：每个 AI 的追问风格不同，覆盖更全
- **录音自己的回答**：听回放比看文字更容易发现表达问题
- **重点练"追问链"**：面试官不会只问表面，他们会连续追问 3-4 层
