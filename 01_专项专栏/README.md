# 🔬 专项专栏

> 按知识域分类的深度练习，每个子目录是一个完整专题

## 📖 这个模块做什么？

这是学习系统的核心内容区域。按照知识域分类，每个子目录覆盖一个完整的技术专题，从入门到面试级别的深度练习。

## 📂 子目录说明

| 目录           | 内容                                                         | Notebook 数量 |
| :------------- | :----------------------------------------------------------- | :-----------: |
| `00_基础/`     | Python/Pandas 基础练习、GroupBy/Lambda/Filter/Pivot 专项训练 |       7       |
| `01_AB试验/`   | A/B 测试基础 → 进阶 → 完整 SOP → CUPED/Delta Method 方差缩减 |       8       |
| `02_机器学习/` | 分类/回归/聚类/NLP/时序预测，含多个完整实战项目              |      30+      |
| `03_归因分析/` | 多触点归因模型、马尔可夫链归因                               |       2       |
| `04_因果推断/` | PSM/DID/RDD/IV/DoWhy/DML，从理论到完整实战                   |      20       |

## 🚀 如何使用？

### 推荐学习路径

```
00_基础 → 01_AB试验 → 02_机器学习(基础) → 03_归因分析 → 04_因果推断
                                    ↓
                            02_机器学习(NLP/时序)
```

### 每个 Notebook 的结构

大部分 Notebook 遵循统一结构：
1. **商业背景** — 为什么要做这个分析
2. **数据准备** — 使用真实数据集（Kaggle）
3. **分级挑战** — Level 1/2/3 层层递进
4. **参考答案** — 折叠隐藏，先自己做再对答案
5. **结论与建议** — 可落地的业务建议（不是只看 R²）

## 💡 来自作者的经验

- **因果推断是最大的加分项**：掌握 PSM/DID 可以让你在面试中碾压大部分候选人
- **NLP 练习要用真实评论数据**：mock 数据无法暴露文本清洗的真正难题
- **每做完一个 Notebook，立刻用 `/update_cheatsheet` 沉淀到速查手册**
