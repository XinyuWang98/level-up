# 🔢 NumPy (数值计算引擎)
*数组运算的地基，Pandas 的底层。*

| 函数 (Function) | 作用 (Action)           | 示例 (Example)                                      |
| :-------------- | :---------------------- | :-------------------------------------------------- |
| `np.array()`    | **基础创建**            | `np.array([1, 2, 3])`                               |
| `np.arange()`   | **等差数列** (左闭右开) | `np.arange(0, 10, 2)` → `[0, 2, 4, 6, 8]`          |
| `np.linspace()` | **等分数列** (包含端点) | `np.linspace(0, 1, 5)` → `[0, 0.25, 0.5, 0.75, 1]` |
| `np.zeros()`    | **全0矩阵**             | `np.zeros((3,3))` (初始化占位)                      |
| `np.ones()`     | **全1矩阵**             | `np.ones((2,2))`                                    |
| `np.where()`    | **条件判断** (IF)       | `np.where(cond, x, y)` → 相当于 Excel 的 `IF`      |
| `np.log()`      | **对数变换**            | 压缩长尾分布 (如收入/销量)                          |

### 核心概念 (Core Concepts)
*   **Broadcasting (广播机制)**: 神奇的自动对齐。即使形状不一样 (如 `[3x3] + [1x3]`)，也能进行加减乘除。
*   **Boolean Indexing**: `arr[arr > 5]` (筛选神器，比循环快100倍)。
